# Photo Portfolio

A responsive web application that empowers users to build their own personalized photography portfolios. Users can seamlessly upload images to their custom profiles and explore a global gallery of work by searching across all photos by title, description, or category.

## Features

* **User profiles.** Create and later log in your own profile
* **Photo uploading.** Upload photos to your profile
* **Search through all photos.** Find photos with the usage of searching by title, descriptoin or category 

## Technologies used

* Flask 3.0.0
* React.js
* MariaDB
